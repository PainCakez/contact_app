# Lastenheft

## Personenbezogene Informationen

* Vorname
* Nachname
* Firma
* Adresse
* Telefonnummer
* E-Mail

## Firmenbezogene Informationen

* Name
* Firmensitz (Adresse)
* Mitarbeiter

## Funktionen

* Sortieren nach Name
* Sortieren nach Firma (Mitarbeiter einer Firma anzeigen)
* Wenn Firma gelöscht wird -> Frage: Sollen Mitarbeiter aus Kontakten gelöscht werden?
